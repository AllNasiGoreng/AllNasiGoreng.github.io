CSS CSS 4 you Eine aktuelle CSS3-Referenz mit ergänzenden Ressourcen, wie zum Beispiel einem Tutorial für Einsteiger, einer Übersicht zur Browserkompatibilität verschiedener CSS-Befehle und verschiedenen Tipps-und-Tricks-Seiten für schöneres CSS.
HTML HTML5 Doctor Eine aktuelle HTML5-Kurzreferenz mit kleinen Abstracts zu verschiedenen HTML5-Elementen. Von dort aus wird immer zur erweiterten Beschreibung auf w3.org verlinkt, sodass bei Bedarf detaillierte Informationen eingeholt werden können. Außerdem ist immer ein Beispielcode vorhanden. Zusätzlich gibt es weitere hilfreiche Ressourcen, wie zum Beispiel Artikel zu Semantik oder Formularen.
Semantisch korrektes HTML w3schools: HTML5 Semantic Elements Gibt eine kurze Übersicht über den Begriff der Semantik in HTML5. Zur Vertiefung können Artikel auf HTML5 Doctor (siehe oben) gelesen werden.
Validierung Validator Hier kann die Validität von HTML5-Dokumenten überprüft werden. Es werden nicht nur Syntax-Fehler gefunden, sondern in vielen Fällen auch ob ein Element an der richtigen Stelle verwendet wird. Der Code kann über die Angabe einer URI, das Hochladen einer Datei, oder das Pasten des Quellcodes überprüft werden.
Webstandard Can I Use Hier kann überprüft werden, ob ein bestimmtes HTML5-Element oder auch ein CSS- oder JavaScript-Befehl, noch aktuell ist, oder ob von der Verwendung abgeraten wird, da er nicht mehr den Webstandards entspricht.
Accessibility w3: Accessibility Gibt eine Übersicht über das Was, Warum und Wie der Accessibility. Zusätzlich wird auf weitere Ressourcen zum Nachlesen von Details und zum umfassenden Einarbeiten in die Materie verlinkt.
Usability NNGroup Usability 101: Introduction to Usability Gibt eine Einführung in das Thema der Usability. Es wird beschrieben was Usability ist und wofür sie wichtig ist. Außerdem gibt es Links zu hilfreichen Ressourcen, mit deren Hilfe man konkret lernen kann, die Usability seiner Webseite zu verbessern.
XML w3schools: XML Beschreibt die Grundlagen von XML und enthält auch eine XML-Referenz
